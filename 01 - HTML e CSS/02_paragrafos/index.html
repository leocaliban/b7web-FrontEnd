Exemplo de um parágrafo de uma linha

Este parágrafo utiliza uma tag 'br'
para quebrar essa linha

        A tag 'pre':
        O texto pré-formatado é um parágrafo demarcado para deixar o texto intacto.

        Deve manter os espaços em branco;
        Os caracteres devem manter a mesma proporção tanto na altura quanto na largura;
        Pode desabilitar a quebra automática de linha;
        Não devem desabilitar o processamento bidirecional de texto.
